This file is only for future reference on how to run this site locally.

To run this locally need to install a few packages:
	sudo apt install ruby ruby-dev make gcc build-essential patch ruby-dev zlib1g-dev liblzma-dev

	sudo gem install jekyll bundler

	cd into correct folder
	bundle install

Command for running the local server can be added as an alias in the bashrc file. The command for running the local server is 'bundle exec jekyll serve'.
